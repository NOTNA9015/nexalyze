How to Join

Our server supports both Java and Bedrock editions with crossplay. Follow these simple steps to join the fun!

Java Edition

1

Open Minecraft

Launch Minecraft Java Edition (version 1.21.4 or newer required)

2

Go to Multiplayer

Click "Multiplayer" from the main menu

3

Add Server

Click "Add Server" and enter these details:

Server Name: Nexalyze
Server Address: mc.nexalyze.dk
4

Join Server

Select the server from your list and click "Join Server"

Bedrock Edition

1

Open Minecraft

Launch Minecraft Bedrock Edition on your device (mobile, console, or Windows 10/11)

2

Go to Servers

Click "Play" then go to the "Servers" tab

3

Add Server

Scroll down and click "Add Server" then enter these details:

Server Name: Nexalyze
Server Address: mc.nexalyze.dk
Port: 19132
4

Join Server

Select the server from your list and click "Play"

Crossplay Information

Our server uses GeyserMC to allow Java and Bedrock players to play together. Some features may work slightly differently between editions.
